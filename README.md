Pregnant Fitness Avatar Application

Description
This project focuses on the creation of a specialized 3D pregnant avatar designed for a fitness application. The goal of the thesis is to develop an interactive platform that provides tailored exercise videos led by a 3D avatar, customized for each trimester of pregnancy.

Tools & Technologies Used
To achieve high realism and correct anatomical proportions, the following pipeline was used:

MakeHuman: Used for the initial base human model generation.

Meshcapade: Leveraged for specialized body modeling and ensuring the avatar reflects the physical changes during pregnancy.

Blender: The primary environment for rigging, skinning, and creating smooth, realistic animations for the exercises.

Unity: The core engine where the avatar and animations are integrated to build the final application.

Key Features
Trimester-Specific Content: Exercises are categorized and adapted based on the specific needs and safety requirements of each pregnancy stage.

3D Animated Guidance: A realistic pregnant avatar demonstrates proper form and technique for various prenatal exercises.

Technical Integration: Seamless workflow from character generation to 3D animation and engine implementation.
